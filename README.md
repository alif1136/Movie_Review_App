# 🎬 Cinephiler – Movie Review & Watchlist App



## 📌 Overview

**Cinephiler** is a modern Flutter-based movie review and tracking application designed for movie lovers who want to explore films, track watched movies, create wishlists, and review their favorite titles.

The app features a clean and cinematic UI inspired by modern streaming platforms, providing users with an engaging browsing experience.

---

# ✨ Features

## 🏠 Home Screen

* Browse newly released movies
* Explore upcoming movies
* Clean and responsive movie cards
* Smooth horizontal scrolling sections

## 🔍 Search Movies

* Search movies instantly
* Find detailed information about films
* Easy navigation and filtering experience

## ❤️ Wishlist System

* Save movies to wishlist
* Manage favorite movies
* Quick access to movies users want to watch later

## 👤 User Profile

* Personalized user profile
* Track watched movie count
* Track total watch time
* Favorite movies section
* Recently watched movies history

## 🎥 Movie Details

* Movie poster and overview
* Release date information
* Runtime details
* Cast & crew section
* Ratings and reviews
* Watch trailer button
* Add to wishlist option
* Watch log tracking

# 🚀 Getting Started

## Prerequisites

Before running the project, make sure you have:

* Flutter SDK installed
* Android Studio / VS Code
* Dart SDK
* Emulator or physical device

---

## Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/cinephiler.git
```

### 2️⃣ Navigate to project folder

```bash
cd cinephiler
```

### 3️⃣ Install dependencies

```bash
flutter pub get
```

### 4️⃣ Run the app

```bash
flutter run
```
